# LearnOS Dashboard

A modern learning dashboard with course tracking, progress visualization, and XP system.

## Features

- 📊 Real-time progress tracking
- 🎓 Multiple courses with XP counters
- 🎨 Dark mode design with smooth animations
- 📱 Responsive layout
- ⚡ Fast page navigation

## Files

- `index.html` - Main dashboard page
- `navigation.js` - Page navigation logic
- `dashboard.js` - Progress animation utilities
- `app.js` - Main application initialization
- `courses.json` - Course data
- `variables.css` - CSS custom properties
- `layout.css` - Layout and component styles

## Local Development

```bash
python -m http.server 8000
```

Visit `http://localhost:8000` in your browser.

## Deployment

This dashboard is ready to deploy to GitHub Pages, Netlify, Vercel, or any static hosting service.

### GitHub Pages
1. Push to GitHub repository
2. Enable GitHub Pages in repository settings
3. Select `main` branch as source

### Netlify
1. Connect your GitHub repository
2. Build command: (leave empty)
3. Publish directory: `.`

### Vercel
1. Import your GitHub repository
2. Deploy!
